DESCRIPTION

r.viewshed.total counts the number of visible cells from each point within the set of input points. After counting it displays them in raster format.

NOTES

The module uses r.viewshed. It requires the same inputs, and can use the same flags.

Overall computation is slow for large or high resolution DEMs. This can somewhat be addressed if the default or a lower max_distance parameter is used.

EXAMPLE

With default parameters and flags -c (consider curvature of the earth) and -r (consider the effect of atmospheric refraction) enabled:
r.viewshed.total -c -r vect=points_example rast=dem_example output=total_viewshed_example

SEE ALSO

r.viewshed, r.viewshed.cva
